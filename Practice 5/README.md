# Practice 5 - Python Regular Expressions (RegEx)

## Objective

This practice focuses on learning Python Regular Expressions using the `re` module.

## Files

- `receipt_parser.py` - parses receipt data from raw.txt
- `raw.txt` - receipt text
- `regex_exercises.py` - 10 Python RegEx exercises
- `README.md` - project description

## Topics Covered

- RegEx introduction
- Metacharacters
- Special sequences
- Sets and character classes
- Quantifiers
- `re.search()`
- `re.findall()`
- `re.split()`
- `re.sub()`
- `re.match()`
- Flags

## Receipt Parser Tasks

The program extracts:

1. All prices
2. Product names
3. Total amount
4. Date and time
5. Payment method
6. Structured JSON output

## How to Run

```bash
python receipt_parser.py
```

```bash
python regex_exercises.py
```
